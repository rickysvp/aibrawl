# AI Brawl - 黑客松项目文档

## 🎯 项目定位

**AI Brawl** 是首个基于 Monad 链的 **AI Agent 自动对战竞技平台**，将 AI 代理与 DeFi 游戏化结合，打造 "Agent 2 Earn" 全新范式。

> 让 AI 为你战斗，让收益自动产生。

---

## ✨ 核心亮点

### 1. 创新机制：Agent 2 Earn
- **铸造 AI Agent**：创建独特的像素风格 Agent NFT
- **自动对战**：10 人实时大乱斗，秒级结算
- **收益掠夺**：击败对手，赢取对方资产
- **7x24 小时**：无需人工干预，持续产生收益

### 2. 经济模型：正向飞轮
```
用户铸造 Agent → 注入资金参与战斗 → 产生手续费 → 
50% 分配给流动性提供者 → 吸引更多流动性 → 更大奖池 → 更多用户参与
```

### 3. 技术亮点
| 特性 | 说明 |
|------|------|
| ⚡ 秒级结算 | 10 秒战斗轮次，即时收益分配 |
| 🎮 实时渲染 | Canvas 驱动的像素风战斗动画 |
| 💧 流动性挖矿 | 质押 MON 代币，分享平台 50% 收益 |
| 📊 数据上链 | 所有战斗记录、收益分配透明可查 |

### 4. 用户体验
- **零门槛**：无需复杂的 DeFi 操作，游戏化界面
- **即时反馈**：实时战斗画面，直观收益展示
- **社交属性**：排行榜、锦标赛、社区对战

---

## 🏗️ 技术架构

### 区块链层
- **Monad 链**：高性能 L1，低 Gas、快速确认
- **智能合约**（规划中）：Agent NFT、战斗结算、收益分配

### 应用层
- **前端**：React 18 + TypeScript + Vite
- **状态管理**：Zustand，实时同步战斗状态
- **动画渲染**：Canvas API，60fps 流畅战斗

### 数据层
- **Supabase**：PostgreSQL + Realtime
- **数据持久化**：Agent 状态、战斗历史、收益记录

## 🎮 使用场景

### 场景 1：被动收益（DeFi 用户）
> 用户质押 MON 代币到流动性池，无需参与战斗，即可获得平台 50% 手续费收益。

**适合人群**：追求稳定收益的 DeFi 投资者

### 场景 2：主动竞技（GameFi 玩家）
> 用户铸造多个 Agent，分配不同策略（高攻击/高防御），参与自动战斗，通过胜率获取收益。

**适合人群**：喜欢策略游戏的玩家

### 场景 3：资产管理（Web3 用户）
> 用户将资产注入 Agent，由 AI 自动参与战斗，实现资产的自动化增值。

**适合人群**：希望资产自动增值的 Holder
